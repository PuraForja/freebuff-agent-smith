## 📋 What is FreeBuff Agent Smith V2?

**FreeBuff Agent Smith V2** is an autonomous AI engineering agent that discovers, analyzes, converts, generates and installs AI agents for the [FreeBuff](https://freebuff.com) ecosystem.

**Em português:**
Um engenheiro de agentes para o ecossistema FreeBuff. Analisa agentes já instalados e repositórios do GitHub para criar, converter e instalar novos agentes compatíveis automaticamente.

## 🤖 Using @agent-smith-v2

After installation, open FreeBuff/Codebuff in your project directory and use:

### Install existing agents
```
@agent-smith-v2 instale python-patterns
@agent-smith-v2 instale error-handling
@agent-smith-v2 instale api-design
```

### Create a new agent
```
@agent-smith-v2 crie um agente para revisão de código Python
@agent-smith-v2 crie um agente para deploy automatizado
```

### Convert agent from another framework
```
@agent-smith-v2 converta este agente do Claude Code para FreeBuff
@agent-smith-v2 adapte este agente para o padrão FreeBuff
```

### List and manage
```
@agent-smith-v2 liste
@agent-smith-v2 atualize
@agent-smith-v2 remova python-patterns
```

## 🧠 RAG for Agent Engineering

FreeBuff Agent Smith V2 implements a **RAG (Retrieval-Augmented Generation)** system for agent engineering. Instead of just answering questions, it:

1. **Retrieves knowledge** from multiple sources:
   - Agents already installed in the project
   - Public GitHub repositories
   - Official FreeBuff documentation
   - Implementations found on the web

2. **Combines and analyzes** all information

3. **Generates a better agent** using all collected context

This means each created agent benefits from accumulated knowledge from diverse sources.
