# Project knowledge — freebuff-ecc-bridge

## What is this project?
Provides [ECC](https://github.com/affaan-m/ECC) (224k+ ⭐) agents and skills for [Freebuff](https://freebuff.com). Agents are TypeScript files in `.agents/`, skills are Markdown in `skills/`.

## Quickstart
- **Install/update agents:** `bash scripts/ecc-install.sh`
- **Update only new:** `bash scripts/ecc-install.sh --update`
- **Force reinstall:** `bash scripts/ecc-install.sh --force`

## Architecture

### Key directories
```
freebuff-ecc-bridge/
├── .agents/              ← 67 TypeScript agents (spawnable with @)
├── skills/               ← 278 Markdown skills (use read_files)
├── CATALOGO.md           ← Complete resource catalog
├── scripts/
│   └── ecc-install.sh    ← Installation/update script
└── knowledge.md          ← This file
```

### Data flow
```
1. ECC (github.com/affaan-m/ECC) — source
   ↓ git clone + ecc-install.sh
2. freebuff-ecc-bridge/
   ↓
3. .agents/*.ts — TypeScript agents
4. skills/*.md — Markdown skills
```

## Conventions

### Agents (.agents/*.ts)
- TypeScript files with AgentDefinition
- Spawnable with `@agent-name`
- Models: mimo/mimo-v2.5 or deepseek/deepseek-v4-flash
- Do NOT edit manually — generated by ecc-install.sh

### Skills (skills/*.md)
- Markdown files with instructions
- Load with `read_files skills/nome.md`
- NOT registered as Codebuff skills

## Gotchas
1. **Run ecc-install.sh** to update agents/skills from ECC
2. **Skills ≠ Codebuff skills** — use `read_files`, not `skill "name"`
3. **Agents are TypeScript** — not markdown reference docs
